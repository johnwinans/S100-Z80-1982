# CPU Card

This board uses a Z80 S-100 bus interface based on that used in the
[Digiac CT-804](https://github.com/johnwinans/digiac-ct-804).

A marked up schematic and notes describing how my board implementation came to be
can be found in:
- [ct-804-markup.pdf](./ct-804-markup.pdf)
- [notes.pdf](./notes.pdf)
